# 評価スケール・効果判定規準 (ECOG PS / Karnofsky / mMRC / RECIST)

臨床で参照頻度の高い評価スケールと効果判定規準を 1 ファイルに集約。数値・閾値・定義を「その場で引く」用途。

## いつ使うか

- 利用者が ECOG PS / Karnofsky (KPS) / mMRC / RECIST、または奏効・CR・PR・SD・PD・標的病変・SOD 等に言及した時
- 各スケールのグレード定義・閾値・判定区分を確認したい時

**自前の記憶で数値を答えず、必ず本SKILLの表を引いて値・定義を提示すること。** 本SKILLは定義の参照のみで、患者データからの自動グレーディングや SOD 計算は行わない (該当値は利用者と確認しながら手で当てはめる)。

出典: ECOG PS は ECOG、Karnofsky PS は原典、mMRC は modified MRC、RECIST 1.1 は Eisenhauer et al. (Eur J Cancer 2009) / JCOG 日本語版。

## ECOG Performance Status (PS)

全身状態の指標。0〜4 の 5 段階(死亡を 5 とする場合もある)。

| Grade | 定義 |
|---|---|
| 0 | 全く問題なく活動できる。発病前と同じ日常生活が制限なく行える |
| 1 | 肉体的に激しい活動は制限されるが、歩行可能で軽作業や座位作業は行える(軽い家事・事務作業など) |
| 2 | 歩行可能で身の回りのことはすべて可能だが、作業はできない。日中の 50% 以上はベッド外で過ごす |
| 3 | 限られた身の回りのことしかできない。日中の 50% 以上をベッドか椅子で過ごす |
| 4 | 全く動けない。身の回りのことは全くできない。完全にベッドか椅子で過ごす |

## Karnofsky Performance Status (KPS)

100〜0 の 11 段階(10 刻み)。

| KPS | 定義 |
|---:|---|
| 100 | 正常、臨床症状なし |
| 90 | 軽い臨床症状はあるが正常活動可能 |
| 80 | かなり臨床症状があるが努力して正常活動可能 |
| 70 | 自分自身の世話はできるが、正常活動・労働は不可能 |
| 60 | 自分に必要なことはできるが、時々介助が必要 |
| 50 | 病状を考慮した看護および定期的な医療行為が必要 |
| 40 | 動けず、適切な医療・看護が必要 |
| 30 | 全く動けず入院が必要だが、死はさしせまっていない |
| 20 | 非常に重症で入院が必要、積極的な治療が必要 |
| 10 | 死期が切迫している |
| 0 | 死 |

## mMRC 息切れスケール (modified Medical Research Council)

呼吸困難の程度。0〜4 の 5 段階。

| Grade | 定義 |
|---|---|
| 0 | 激しい運動をした時だけ息切れがある |
| 1 | 平坦な道を急ぎ足で歩く、またはゆるやかな上り坂を歩く時に息切れがある |
| 2 | 息切れのため同年代より平坦な道を歩くのが遅い、または自分のペースで平坦な道を歩いていても息継ぎのため立ち止まることがある |
| 3 | 平坦な道を約 100 m、または数分歩くと息継ぎのため立ち止まる |
| 4 | 息切れがひどく家から出られない、または衣服の着替えでも息切れがある |

## RECIST 1.1 (固形がん治療効果判定規準)

### 測定可能病変

- 測定可能病変: CT で長径 **≥10 mm**(スライス厚 ≤5 mm)。リンパ節は **短径 ≥15 mm**
- 測定不能病変: 上記未満の小病変、リンパ節短径 10〜15 mm、骨病変・嚢胞性病変・髄膜病変・癌性リンパ管症 等

### 標的病変 (target lesion)

- 全病変から最大 **5 個**(1 臓器あたり 2 個まで)を選ぶ
- **径の和 (SOD; sum of diameters)** を測る(リンパ節は短径を加算)

### 効果判定(標的病変)

| 判定 | 定義 |
|---|---|
| CR(完全奏効) | 全標的病変が消失。病的リンパ節は短径 <10 mm まで縮小 |
| PR(部分奏効) | SOD がベースラインから **≥30% 減少** |
| PD(進行) | SOD が経過中の最小値 (nadir) から **≥20% 増加**、かつ絶対値で **≥5 mm 増加**。または新病変出現 |
| SD(安定) | PR にも PD にも該当しない |

### 非標的病変

CR / non-CR・non-PD / PD の 3 区分で評価。

### 総合効果

標的病変・非標的病変・新病変の有無を組み合わせて判定する。確定 (confirmed) 効果は一定間隔(通常 4 週以上)を空けた再評価で確認する。
